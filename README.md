Esse programa é um gerenciador simples de contatos para uso no terminal, criado como exercício de lógica de programação.

Funcionalidades principais:

cadastrar contato com ID gerado automaticamente, nome, função, DDD (2 dígitos) e telefone (9 dígitos);
validação de DDD e telefone para aceitar somente números no formato esperado;
consultar contatos:
listar todos os contatos cadastrados;
buscar por ID;
buscar por função (texto, sem diferenciar maiúsculas/minúsculas);
remover contato pelo ID.
Como funciona:

os contatos são mantidos em uma lista lista_contatos em memória;
ao encerrar o programa, os dados são perdidos;
o programa apresenta um menu interativo com opções e trata entradas inválidas para evitar falha abrupta.
Para executar:

abra o terminal na pasta do projeto;
rode o código;
escolha as opções 1 a 4 no menu principal.
